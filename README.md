# ВОЛКОВСКИЙ ГОК - Система документации

Веб-приложение для управления документацией и инструкциями пользователя для проекта ВОЛКОВСКИЙ ГОК.

## Возможности

- Просмотр описательных карточек экранов системы
- Пошаговые инструкции с скриншотами
- Информационные блоки "О системе" и "Особенности системы"
- Редактор для добавления и управления карточками
- Хранение данных в JSON формате

## Установка

```bash
npm install
```

## Запуск

```bash
node server.js
```

Откройте в браузере: http://localhost:3000 (окно пользователя)  или http://localhost:3000/admin.html (окно администратора)

## Структура проекта

```
volk_gok_docs_v2_json/
├── admin.html      # Документация с редактором
├── server.js       # Node.js сервер
├── data.json       # Данные карточек (JSON)
├── view.html       # Документация без редактора
└── screenshots/    # Папка со скриншотами
```


## Использовано

- Node.js + Express
- Vanilla JavaScript
- CSS3
